AI Health Assistant — Frontend

Overview:
- Modern responsive React frontend using Tailwind, Three.js background, and GSAP animations.

Quick start:

1. Install dependencies

```bash
cd frontend
npm install
```

2. Run dev server

```bash
npm run dev
```

3. Build for production

```bash
npm run build
npm run preview
```

Notes:
- GSAP is used for nav and micro-animations. Three.js background is optimized for mobile (reduced particles and lower detail).
- For cross-platform mobile/desktop look, this app is responsive; to package as native desktop/mobile consider Electron or Capacitor.
